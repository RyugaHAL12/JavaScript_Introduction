//  JavaScriptとは・・・  //
→
    プログラミング言語の一つで、ブラウザーで動く言語として開発された。
    
    CSSやHTMLでは実装できないことを実装するのがJavaScript。
    例えば...
    Webサイトにアニメーションをつけたり
    リアルタイムでWebサイトを書き換えたり
    ユーザーの操作に合わせて、動きや機能を加えたりする。

    JavaScriptは初心者にもおすすめ　←個人的には大嘘。中身を知らなければおすすめ。

//  JavaScriptでできること  //
→
    条件分岐でHTMLやCSSを書き換えることができる。

    ・テキストの書き換え
    →
        メニューを開く→閉じる表示に変更→開くに変更

        読み込み中→完了に変更

    ・画像の変更
    →
        画像にマウスカーソルを乗せる→犬の画像が猫の画像に変化
    
    ・色の変化
    →
        ライトモード時：白背景→ダークモード時：黒背景

//  Tips  //
→
    ・効率的な学習方法は？
    →
        ハンズオン(実際に手を動かす)方式でサンプルデータやデモを参考に
        自分でJavaScriptなどのコードを組んでみて
        書いたコードが何を意味しているのかを一つ一つ追って行くことで
        少しずつ、何をやっているのかがわかってくる。

    ・Webデザイナーでも楽しく学べるJavaScript本
    →
        1冊ですべて身につくJavaScript入門講座
        これがいいらしい。(ダイマ)
